## 1. イベントモデルの全体像

GoJS には 3 レベルのイベント仕組みがあります。

```
┌───────────────────────────────────────────────┐
│ レベル 1: GraphObject イベント                  │
│  → クリック、ホバー、ドラッグなど               │
│  → ノード、リンク、Shape 等に直接設定           │
├───────────────────────────────────────────────┤
│ レベル 2: DiagramEvent                         │
│  → 選択変更、リンク作成、テキスト編集完了など    │
│  → diagram.addDiagramListener() で登録         │
├───────────────────────────────────────────────┤
│ レベル 3: Model Changed リスナー                │
│  → モデルデータの変更を検知                     │
│  → model.addChangedListener() で登録           │
└───────────────────────────────────────────────┘
```

## 2. GraphObject イベント（クリック等）

### 2.1 利用可能なイベント一覧

| イベント | 発火タイミング | 引数 |
|---|---|---|
| `click` | シングルクリック | `(e: InputEvent, obj: GraphObject)` |
| `doubleClick` | ダブルクリック | `(e: InputEvent, obj: GraphObject)` |
| `contextClick` | 右クリック | `(e: InputEvent, obj: GraphObject)` |
| `mouseEnter` | マウス侵入 | `(e: InputEvent, obj: GraphObject, prev: GraphObject)` |
| `mouseLeave` | マウス退出 | `(e: InputEvent, obj: GraphObject, next: GraphObject)` |
| `mouseOver` | マウス移動中 | `(e: InputEvent, obj: GraphObject)` |
| `mouseDragEnter` | ドラッグ侵入 | `(e: InputEvent, obj: GraphObject, prev: GraphObject)` |
| `mouseDragLeave` | ドラッグ退出 | `(e: InputEvent, obj: GraphObject, next: GraphObject)` |
| `mouseDrop` | ドロップ | `(e: InputEvent, obj: GraphObject)` |
| `mouseHold` | 長押し | `(e: InputEvent, obj: GraphObject)` |

### 2.2 クリックイベント

```typescript
import * as go from "gojs";

diagram.nodeTemplate = new go.Node("Auto", {
  // シングルクリック
  click: (e: go.InputEvent, node: go.GraphObject) => {
    const n = node as go.Node;
    console.log("クリック:", n.data.name);
  },

  // ダブルクリック
  doubleClick: (e: go.InputEvent, node: go.GraphObject) => {
    const n = node as go.Node;
    console.log("ダブルクリック:", n.data.name);
    // 例: 詳細ダイアログを開く
    openDetailDialog(n.data);
  },

  // 右クリック
  contextClick: (e: go.InputEvent, node: go.GraphObject) => {
    const n = node as go.Node;
    console.log("右クリック:", n.data);
  },
}).add(
  new go.Shape("RoundedRectangle", { fill: "#2196F3", stroke: null }),
  new go.TextBlock({ margin: 10, stroke: "white" }).bind("text", "name")
);
```

### 2.3 ホバーイベント（マウスオーバー・アウト）

```typescript
diagram.nodeTemplate = new go.Node("Auto", {
  // マウスが入った時
  mouseEnter: (e: go.InputEvent, node: go.GraphObject) => {
    const shape = (node as go.Node).findObject("SHAPE") as go.Shape;
    if (shape) {
      shape.fill = "#1976D2";       // ホバー色
      shape.strokeWidth = 2;
    }
    // カーソル変更
    (node as go.Node).cursor = "pointer";
  },

  // マウスが出た時
  mouseLeave: (e: go.InputEvent, node: go.GraphObject) => {
    const shape = (node as go.Node).findObject("SHAPE") as go.Shape;
    if (shape) {
      shape.fill = node.part?.data.color ?? "#2196F3"; // 元の色に戻す
      shape.strokeWidth = 1;
    }
  },
}).add(
  new go.Shape("RoundedRectangle", {
    name: "SHAPE",   // ← findObject で検索するための名前
    fill: "#2196F3",
    stroke: "#1565C0",
    strokeWidth: 1,
  }).bind("fill", "color"),
  new go.TextBlock({ margin: 10, stroke: "white" }).bind("text")
);
```

## 3. DiagramEvent（ダイアグラムイベント）

### 3.1 主要な DiagramEvent 一覧

| イベント名 | 発火タイミング |
|---|---|
| `"ChangedSelection"` | 選択が変更された |
| `"ObjectSingleClicked"` | オブジェクトがシングルクリックされた |
| `"ObjectDoubleClicked"` | オブジェクトがダブルクリックされた |
| `"ObjectContextClicked"` | オブジェクトが右クリックされた |
| `"BackgroundSingleClicked"` | 背景がシングルクリックされた |
| `"BackgroundDoubleClicked"` | 背景がダブルクリックされた |
| `"LinkDrawn"` | 新しいリンクが描画された |
| `"LinkRelinked"` | リンクが付け替えられた |
| `"SelectionMoved"` | 選択中の要素が移動された |
| `"SelectionCopied"` | 選択中の要素がコピーされた |
| `"SelectionDeleting"` | 選択中の要素が削除される直前 |
| `"SelectionDeleted"` | 選択中の要素が削除された |
| `"TextEdited"` | テキスト編集が完了した |
| `"PartCreated"` | ノードまたはグループが作成された |
| `"PartResized"` | パーツがリサイズされた |
| `"PartRotated"` | パーツが回転された |
| `"ExternalObjectsDropped"` | パレットからドロップされた |
| `"ClipboardPasted"` | クリップボードから貼り付けされた |
| `"DocumentBoundsChanged"` | ドキュメント全体の範囲が変更された |
| `"ViewportBoundsChanged"` | ビューポートが変更された（スクロール/ズーム） |
| `"InitialLayoutCompleted"` | 初回レイアウト計算が完了した |
| `"LayoutCompleted"` | レイアウト計算が完了した |
| `"TreeExpanded"` | ツリーノードが展開された |
| `"TreeCollapsed"` | ツリーノードが折りたたまれた |

### 3.2 使用例

```typescript
// ━━━ 選択変更の検知 ━━━
diagram.addDiagramListener("ChangedSelection", (e: go.DiagramEvent) => {
  const selectedNodes = diagram.selection
    .filter((part) => part instanceof go.Node)
    .toArray();

  console.log("選択中ノード:", selectedNodes.map((n) => n.data));

  // React の状態更新等
  // setSelectedItems(selectedNodes.map(n => n.data));
});

// ━━━ リンク作成の検知 ━━━
diagram.addDiagramListener("LinkDrawn", (e: go.DiagramEvent) => {
  const link = e.subject as go.Link;
  console.log("新しいリンク:", {
    from: link.fromNode?.data,
    to: link.toNode?.data,
  });
});

// ━━━ 削除前の確認 ━━━
diagram.addDiagramListener("SelectionDeleting", (e: go.DiagramEvent) => {
  const count = diagram.selection.count;
  if (!confirm(`${count}件の要素を削除しますか？`)) {
    e.diagram.currentTool.doCancel(); // 削除をキャンセル
  }
});

// ━━━ 初回レイアウト完了後の処理 ━━━
diagram.addDiagramListener("InitialLayoutCompleted", (e: go.DiagramEvent) => {
  // 全体が見えるようにズーム調整
  diagram.zoomToFit();
});
```

## 4. Model Changed リスナー

モデルデータの変更を低レベルで検知します。

```typescript
// ━━━ モデル変更リスナー ━━━
diagram.model.addChangedListener((e: go.ChangedEvent) => {
  // トランザクション完了時のみ処理
  if (e.isTransactionFinished) {
    console.log("トランザクション完了");
    // ここで自動保存等を実行
    autoSave();
    return;
  }

  // 個別の変更を検知
  switch (e.change) {
    case go.ChangeType.Property:
      console.log(`プロパティ変更: ${e.propertyName}`, {
        oldValue: e.oldValue,
        newValue: e.newValue,
      });
      break;

    case go.ChangeType.Insert:
      console.log("データ追加:", e.newValue);
      break;

    case go.ChangeType.Remove:
      console.log("データ削除:", e.oldValue);
      break;
  }
});
```

### Transaction 単位での変更検知

```typescript
diagram.model.addChangedListener((e: go.ChangedEvent) => {
  if (!e.isTransactionFinished) return;

  // Transaction 内のすべての変更を取得
  const txn = e.object as go.Transaction;
  if (!txn) return;

  const changes = txn.changes;
  changes.each((change: go.ChangedEvent) => {
    if (change.modelChange === "nodeDataArray") {
      if (change.change === go.ChangeType.Insert) {
        console.log("ノード追加:", change.newValue);
      }
      if (change.change === go.ChangeType.Remove) {
        console.log("ノード削除:", change.oldValue);
      }
    }
  });
});
```

## 5. 選択イベント

### 5.1 選択の操作と監視

```typescript
// ━━━ 選択を監視 ━━━
diagram.addDiagramListener("ChangedSelection", (e: go.DiagramEvent) => {
  const sel = diagram.selection;

  // 選択されたノード
  const nodes: go.ObjectData[] = [];
  sel.each((part) => {
    if (part instanceof go.Node) {
      nodes.push(part.data);
    }
  });

  console.log(`${nodes.length}件のノードを選択中`);
  updateSidebar(nodes); // サイドバーを更新
});

// ━━━ プログラムから選択する ━━━
function selectNode(key: number): void {
  const node = diagram.findNodeForKey(key);
  if (node) {
    diagram.select(node);
    diagram.centerRect(node.actualBounds); // 画面中央に移動
  }
}

// ━━━ 複数選択 ━━━
function selectNodes(keys: number[]): void {
  diagram.clearSelection();
  keys.forEach((key) => {
    const node = diagram.findNodeForKey(key);
    if (node) node.isSelected = true;
  });
}

// ━━━ 選択可能条件を制限 ━━━
diagram.nodeTemplate = new go.Node("Auto", {
  // 特定条件で選択不可
  selectable: true,
  selectionChanged: (node: go.Part) => {
    // 選択状態が変化した時（個別ノード）
    const n = node as go.Node;
    console.log(`${n.data.name}: ${n.isSelected ? "選択" : "選択解除"}`);
  },
}).add(
  new go.Shape("RoundedRectangle").bind("fill", "color"),
  new go.TextBlock({ margin: 10 }).bind("text", "name")
);
```

## 6. テキスト編集イベント

```typescript
// ━━━ テキスト編集の検知 ━━━
diagram.addDiagramListener("TextEdited", (e: go.DiagramEvent) => {
  const textBlock = e.subject as go.TextBlock;
  const oldText = e.parameter as string; // 編集前のテキスト

  console.log(`テキスト変更: "${oldText}" → "${textBlock.text}"`);

  // バリデーション（必要に応じてロールバック）
  if (textBlock.text.trim() === "") {
    diagram.startTransaction("revert text");
    textBlock.text = oldText;
    diagram.commitTransaction("revert text");
    alert("テキストを空にすることはできません");
  }
});

// ━━━ TextEditingTool のカスタマイズ ━━━
const textEditingTool = diagram.toolManager.textEditingTool;

// 編集開始条件を変更（デフォルトはダブルクリックまたは F2）
textEditingTool.starting = go.TextEditingStarting.SingleClick;

// バリデーション関数
textEditingTool.textValidation = (
  textBlock: go.TextBlock,
  oldString: string,
  newString: string
): boolean => {
  // 50文字以内のみ許可
  if (newString.length > 50) {
    alert("50文字以内で入力してください");
    return false;
  }
  // 空文字を禁止
  if (newString.trim() === "") {
    return false;
  }
  return true;
};
```

```typescript
    if (shape) shape.stroke = "#2196F3";
  },

    const shape = (node as go.Node).findObject("SHAPE") as go.Shape;

    // ドロップされた

    const dropped = diagram.selection.first();

      console.log(`${dropped.data.name} を ${target.data.name} にドロップ`);
    }
  },
}).add(
  new go.Shape("RoundedRectangle", { name: "SHAPE", fill: "white", stroke: "#999" }),
  new go.TextBlock({ margin: 10 }).bind("text", "name")
);
```

## 8. リンクバリデーション

リンク接続の可否をプログラムで制御できます。

```typescript
// ━━━ リンク作成のバリデーション ━━━
diagram.toolManager.linkingTool.linkValidation = (
  fromNode: go.Node,
  fromPort: go.GraphObject,
  toNode: go.Node,
  toPort: go.GraphObject,
  link: go.Link
): boolean => {
  // 自分自身へのリンクを禁止
  if (fromNode === toNode) {
    return false;
  }

  // 同じペアの重複リンクを禁止
  const existingLinks = fromNode.findLinksTo(toNode);
  if (existingLinks.count > 0) {
    return false;
  }

  // カテゴリ制限（例: "input" → "process" のみ許可）
  const fromCat = fromNode.data.category;
  const toCat = toNode.data.category;
  const allowedConnections: Record<string, string[]> = {
    input: ["process"],
    process: ["process", "output"],
    output: [],
  };
  if (!allowedConnections[fromCat]?.includes(toCat)) {
    return false;
  }

  return true;
};

// ━━━ ポート設定（接続点） ━━━
diagram.nodeTemplate = new go.Node("Spot", {
  locationSpot: go.Spot.Center,
}).add(
  // メイン
  new go.Shape("RoundedRectangle", { fill: "#2196F3", stroke: null }),
  new go.TextBlock({ margin: 10, stroke: "white" }).bind("text"),

  // 入力ポート（左）
  new go.Shape("Circle", {
    alignment: go.Spot.Left,
    desiredSize: new go.Size(12, 12),
    fill: "#4CAF50",
    stroke: null,
    portId: "in",
    fromLinkable: false,
    toLinkable: true,
    cursor: "pointer",
  }),

  // 出力ポート（右）
  new go.Shape("Circle", {
    alignment: go.Spot.Right,
    desiredSize: new go.Size(12, 12),
    fill: "#FF5722",
    stroke: null,
    portId: "out",
    fromLinkable: true,
    toLinkable: false,
    cursor: "pointer",
  })
);
```

```typescript
// ━━━ CommandHandler をオーバーライド ━━━
class CustomCommandHandler extends go.CommandHandler {
  // Ctrl+S で保存
  canSave(): boolean { return true; }
  save(): void {
    const json = this.diagram.model.toJson();
    localStorage.setItem("diagram", json);
    console.log("✅ 保存しました");
  }
}

const diagram = new go.Diagram(divId, {
  commandHandler: new CustomCommandHandler(),
});

// ━━━ キーバインドの追加 ━━━
diagram.commandHandler.doKeyDown = function () {
  const e = this.diagram.lastInput;

  // Ctrl+S: 保存
  if (e.control && e.key === "S") {
    e.handled = true;
    (this as CustomCommandHandler).save();
    return;
  }

  // Ctrl+D: 選択ノードの複製
  if (e.control && e.key === "D") {
    e.handled = true;
    this.copySelection();
    this.pasteSelection();
    return;
  }

  // Ctrl+F: 検索ダイアログ（独自実装）
  if (e.control && e.key === "F") {
    e.handled = true;
    openSearchDialog();
    return;
  }

  // デフォルト処理
  go.CommandHandler.prototype.doKeyDown.call(this);
};
```

## 11. 実践パターン集

### 11.1 自動保存（デバウンス付き）

```typescript
let saveTimer: ReturnType<typeof setTimeout> | null = null;

function setupAutoSave(diagram: go.Diagram, delayMs = 1000): void {
  diagram.model.addChangedListener((e: go.ChangedEvent) => {
    if (!e.isTransactionFinished) return;

    // デバウンス: 最後の変更から delayMs 後に保存
    if (saveTimer) clearTimeout(saveTimer);
    saveTimer = setTimeout(() => {
      const json = diagram.model.toJson();
      localStorage.setItem("autosave", json);
      console.log("💾 自動保存完了:", new Date().toLocaleTimeString());
    }, delayMs);
  });
}
```

### 11.2 変更履歴の表示

```typescript
interface ChangeRecord {

}

const changeHistory: ChangeRecord[] = [];

function setupChangeHistory(diagram: go.Diagram): void {
  diagram.model.addChangedListener((e: go.ChangedEvent) => {
    if (!e.isTransactionFinished) return;

    const txn = e.object as go.Transaction;
    if (!txn) return;

    changeHistory.push({
      timestamp: new Date().toLocaleTimeString(),
      description: txn.name,
    });

    // 直近50件のみ保持
    if (changeHistory.length > 50) {
      changeHistory.splice(0, changeHistory.length - 50);
    }

    updateHistoryPanel(changeHistory);
  });
}
```

### 11.3 ノード検索＆ハイライト

```typescript
function searchAndHighlight(diagram: go.Diagram, query: string): void {
  diagram.startTransaction("highlight");

  // 全ノードのハイライトをリセット
  diagram.nodes.each((node) => {
    diagram.model.set(node.data, "isHighlighted", false);
  });

  if (query.trim() === "") {
    diagram.commitTransaction("highlight");
    return;
  }

  const lowerQuery = query.toLowerCase();

  diagram.nodes.each((node) => {
    const text = (node.data.text || node.data.name || "").toLowerCase();
    if (text.includes(lowerQuery)) {
      diagram.model.set(node.data, "isHighlighted", true);
    }
  });

  diagram.commitTransaction("highlight");

  // 最初のマッチにスクロール
  const firstMatch = diagram.nodes.filter(
    (n) => n.data.isHighlighted
  ).first();
  if (firstMatch) {
    diagram.centerRect(firstMatch.actualBounds);
  }
}

// テンプレート側でハイライト対応
new go.Shape("RoundedRectangle")
  .bind("stroke", "isHighlighted", (h: boolean) =>
    h ? "#FF5722" : "#999"
  )
  .bind("strokeWidth", "isHighlighted", (h: boolean) =>
    h ? 3 : 1
  )
```

```typescript
function setupZoomControls(diagram: go.Diagram): void {
  // ズームイン
  const zoomIn = () => {
    diagram.commandHandler.increaseZoom(1.2);
  };

  // ズームアウト
  const zoomOut = () => {
    diagram.commandHandler.decreaseZoom(0.8);
  };

  // 全体表示
  const zoomToFit = () => {
    diagram.zoomToFit();
  };

  // 100% に戻す
  const resetZoom = () => {
    diagram.scale = 1.0;
  };

  // ズーム範囲制限
  diagram.minScale = 0.3;
  diagram.maxScale = 3.0;

  // ビューポート変更イベント
  diagram.addDiagramListener("ViewportBoundsChanged", () => {
    const zoomPercent = Math.round(diagram.scale * 100);
    document.getElementById("zoom-level")!.textContent = `${zoomPercent}%`;
  });
}
```
