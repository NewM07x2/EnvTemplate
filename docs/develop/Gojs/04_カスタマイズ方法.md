## 1. Shape のカスタマイズ

### 1.1 基本プロパティ

| プロパティ | 型 | 説明 | 例 |
|---|---|---|---|
| `figure` | `string` | 図形の種類 | `"Rectangle"`, `"Circle"`, `"Diamond"` |
| `fill` | `string` | 背景色 | `"#4CAF50"`, `"transparent"` |
| `stroke` | `string` | 枠線の色 | `"#333"`, `null`（枠線なし） |
| `strokeWidth` | `number` | 枠線の太さ | `1`, `2`, `0` |
| `strokeDashArray` | `number[]` | 破線パターン | `[6, 3]`（点線） |
| `desiredSize` | `Size` | 固定サイズ | `new go.Size(80, 40)` |
| `minSize` | `Size` | 最小サイズ | `new go.Size(50, 30)` |
| `maxSize` | `Size` | 最大サイズ | `new go.Size(200, NaN)` |
| `opacity` | `number` | 不透明度 (0〜1) | `0.8` |
| `angle` | `number` | 回転角度（度） | `45` |
| `parameter1` | `number` | 図形パラメータ1（角丸半径等） | `10` |

### 1.2 利用可能な Figure 一覧（抜粋）

```typescript
// ━━━ 基本図形 ━━━
"Rectangle"          // 四角形
"RoundedRectangle"   // 角丸四角形
"Circle"             // 円
"Ellipse"            // 楕円
"Diamond"            // ひし形
"Triangle"           // 三角形
"TriangleDown"       // 下向き三角形

// ━━━ フローチャート用 ━━━
"Parallelogram1"     // 平行四辺形（入出力）
"Trapezoid"          // 台形
"Cylinder1"          // シリンダー（データベース）
"Document"           // 書類
"ManualOperation"    // 手動操作
"Terminator"         // 端子

// ━━━ その他 ━━━
"Star"               // 星形
"Pentagon"           // 五角形
"Hexagon"            // 六角形
"Octagon"            // 八角形
"Cloud"              // 雲
"Heart"              // ハート
"Lightning"          // 稲妻
"Arrow"              // 矢印
"Gear"               // 歯車
```

### 1.3 使用例

```typescript
// 角丸+破線+半透明
new go.Shape("RoundedRectangle", {
  fill: "rgba(33, 150, 243, 0.3)",
  stroke: "#1565C0",
  strokeWidth: 2,
  strokeDashArray: [6, 3],
  parameter1: 12,       // 角丸の半径
  desiredSize: new go.Size(120, 60),
})

// データベースアイコン風
new go.Shape("Cylinder1", {
  fill: "#FF9800",
  stroke: "#E65100",
  strokeWidth: 2,
  desiredSize: new go.Size(60, 50),
})
```

## 2. TextBlock のカスタマイズ

### 2.1 基本プロパティ

| プロパティ | 型 | 説明 | 例 |
|---|---|---|---|
| `text` | `string` | 表示テキスト | `"Hello"` |
| `font` | `string` | CSS フォント指定 | `"bold 14px sans-serif"` |
| `stroke` | `string` | テキスト色 | `"#333"` |
| `textAlign` | `string` | 水平配置 | `"center"`, `"left"`, `"right"` |
| `verticalAlignment` | `Spot` | 垂直配置 | `go.Spot.Center` |
| `wrap` | `Wrap` | テキスト折り返し | `go.Wrap.Fit`, `go.Wrap.DesiredSize` |
| `overflow` | `Overflow` | はみ出し処理 | `go.Overflow.Ellipsis` |
| `maxLines` | `number` | 最大行数 | `2` |
| `maxSize` | `Size` | 最大サイズ | `new go.Size(150, NaN)` |
| `editable` | `boolean` | テキスト編集可能 | `true` |
| `isMultiline` | `boolean` | 複数行入力 | `true` |
| `background` | `string` | 背景色 | `"#FFEB3B"` |

### 2.2 使用例

```typescript
// 省略表示付きテキスト
new go.TextBlock({
  font: "13px sans-serif",
  stroke: "#333",
  maxSize: new go.Size(120, NaN),
  maxLines: 2,
  overflow: go.Overflow.Ellipsis,
  wrap: go.Wrap.Fit,
}).bind("text")

// 編集可能テキスト（双方向バインド）
new go.TextBlock({
  font: "bold 14px sans-serif",
  editable: true,
  isMultiline: false,
}).bind(new go.Binding("text").makeTwoWay())
```

## 3. Panel のレイアウト種類

Panel はノード内の要素を配置するコンテナです。

### 3.1 Panel タイプ一覧

| パネルタイプ | 説明 | 用途 |
|---|---|---|
| `"Position"` | 絶対座標で配置 | 自由配置 |
| `"Horizontal"` | 水平方向に並べる | アイコン + テキスト |
| `"Vertical"` | 垂直方向に並べる | タイトル + 説明 |
| `"Auto"` | 最初の子を背景に使用 | Shape + TextBlock |
| `"Spot"` | 相対位置で配置（Spot 指定） | バッジ・アイコン重ね |
| `"Table"` | テーブルレイアウト（行×列） | カード・ER図 |
| `"Viewbox"` | 子を拡大/縮小して合わせる | アイコン表示 |
| `"Grid"` | グリッド背景用 | 背景の罫線 |

### 3.2 Spot パネル（バッジ付きノード）

```typescript
new go.Node("Spot").add(
  // メイン（中央）
  new go.Shape("RoundedRectangle", {
    fill: "#2196F3", stroke: null,
    desiredSize: new go.Size(100, 50),
  }),
  new go.TextBlock({
    font: "bold 14px sans-serif",
    stroke: "white",
  }).bind("text"),

  // バッジ（右上）
  new go.Panel("Auto", {
    alignment: go.Spot.TopRight,
    alignmentFocus: go.Spot.Center,
  }).add(
    new go.Shape("Circle", {
      fill: "#F44336", stroke: null,
      desiredSize: new go.Size(22, 22),
    }),
    new go.TextBlock({
      font: "bold 11px sans-serif",
      stroke: "white",
    }).bind("text", "count")
  ),

  // ステータスアイコン（左下）
  new go.Shape("Circle", {
    alignment: go.Spot.BottomLeft,
    alignmentFocus: go.Spot.Center,
    desiredSize: new go.Size(14, 14),
    stroke: "white", strokeWidth: 2,
  }).bind("fill", "status", (s: string) =>
    s === "online" ? "#4CAF50" : "#9E9E9E"
  )
)
```

### 3.3 Table パネル（カード型ノード）

```typescript
new go.Node("Auto").add(
  new go.Shape("RoundedRectangle", {
    fill: "white", stroke: "#ddd", strokeWidth: 1,
  }),
  new go.Panel("Table", {
    margin: 6,
    defaultAlignment: go.Spot.Left,
  }).add(
    // ── Row 0: アバター + 名前 ──
    new go.Shape("Circle", {
      row: 0, column: 0,
      desiredSize: new go.Size(36, 36),
      fill: "#E3F2FD", stroke: null,
    }),
    new go.TextBlock({
      row: 0, column: 1,
      margin: new go.Margin(0, 0, 0, 8),
      font: "bold 14px sans-serif",
    }).bind("text", "name"),

    // ── Row 1: メールアドレス ──
    new go.TextBlock({
      row: 1, column: 0, columnSpan: 2,
      margin: new go.Margin(4, 0, 0, 0),
      font: "12px sans-serif",
      stroke: "#666",
    }).bind("text", "email"),

    // ── Row 2: 部署 ──
    new go.TextBlock({
      row: 2, column: 0, columnSpan: 2,
      margin: new go.Margin(2, 0, 0, 0),
      font: "11px sans-serif",
      stroke: "#999",
    }).bind("text", "department")
  )
)
```

## 4. ノードテンプレートの設計パターン

### 4.1 テンプレートマップ（カテゴリ別テンプレート）

```typescript
diagram.nodeTemplateMap = new go.Map<string, go.Node>();

// デフォルトテンプレート
diagram.nodeTemplateMap.add("", defaultNodeTemplate);

// カテゴリ別
diagram.nodeTemplateMap.add("Server", serverTemplate);

// { key: 1, text: "Web API", category: "Server" }
```

### 4.2 条件付きスタイリング

```typescript
// ステータスに応じた色分け
new go.Shape("RoundedRectangle")
  .bind("fill", "status", (status: string) => {
    const colorMap: Record<string, string> = {
      active:   "#C8E6C9",
      warning:  "#FFF9C4",
      error:    "#FFCDD2",
      disabled: "#F5F5F5",
    };
    return colorMap[status] ?? "#FFFFFF";
  })
  .bind("stroke", "status", (status: string) => {
    const colorMap: Record<string, string> = {
      active:   "#4CAF50",
      warning:  "#FFC107",
      error:    "#F44336",
      disabled: "#9E9E9E",
    };
    return colorMap[status] ?? "#CCCCCC";
  });
```

## 5. リンクのカスタマイズ

### 5.1 基本プロパティ

| プロパティ | 型 | 説明 |
|---|---|---|
| `routing` | `Routing` | 経路方式（`Normal`, `Orthogonal`, `AvoidsNodes`） |
| `curve` | `Curve` | 曲線の種類（`None`, `Bezier`, `JumpOver`, `JumpGap`） |
| `corner` | `number` | 直角の角丸半径 |
| `fromSpot` | `Spot` | 開始点の位置（`go.Spot.Right`, `go.Spot.Bottom` 等） |
| `toSpot` | `Spot` | 終了点の位置 |
| `toShortLength` | `number` | 矢印分のショートニング距離 |
| `relinkableFrom` | `boolean` | 始点のリンク付け替え可能 |
| `relinkableTo` | `boolean` | 終点のリンク付け替え可能 |
| `reshapable` | `boolean` | 中間点のドラッグ変形可能 |

### 5.2 矢印の種類（Arrow 一覧）

```typescript
// 終端矢印
{ toArrow: "Standard" }      // ▶ 標準三角
{ toArrow: "OpenTriangle" }  // △ 開いた三角
{ toArrow: "Circle" }        // ● 丸
{ toArrow: "Diamond" }       // ◆ ダイヤ
{ toArrow: "BackwardOpenTriangle" } // 逆向き開き三角
{ toArrow: "StretchedDiamond" }     // 横長ダイヤ

// 始端矢印
{ fromArrow: "Backward" }    // 始端に矢印を付ける
```

### 5.3 リンクラベル

```typescript
diagram.linkTemplate = new go.Link({
  routing: go.Routing.Orthogonal,
  corner: 8,
}).add(
  // 線
  new go.Shape({ strokeWidth: 2, stroke: "#555" }),
  // 終端矢印
  new go.Shape({ toArrow: "Standard", fill: "#555", stroke: null }),

  // ━━━ 中央ラベル ━━━
  new go.Panel("Auto").add(
    new go.Shape("RoundedRectangle", {
      fill: "white",
      stroke: "#aaa",
      strokeWidth: 1,
    }),
    new go.TextBlock({
      margin: new go.Margin(2, 6),
      font: "11px sans-serif",
    }).bind("text", "label")
  ),

  // ━━━ 始点側ラベル ━━━
  new go.TextBlock({
    segmentIndex: 0,
    segmentFraction: 0.2,
    font: "bold 11px sans-serif",
    stroke: "#333",
  }).bind("text", "fromLabel"),

  // ━━━ 終点側ラベル ━━━
  new go.TextBlock({
    segmentIndex: -1,
    segmentFraction: 0.8,
    font: "bold 11px sans-serif",
    stroke: "#333",
  }).bind("text", "toLabel"),
);
```

## 6. データバインディング詳細

### 6.1 単方向バインド（Model → View）

```typescript
// 基本
new go.TextBlock().bind("text", "name")

// 変換関数付き
new go.TextBlock().bind("text", "score", (s: number) => `${s}点`)

// プロパティ名が同じ場合は省略可
new go.TextBlock().bind("text")
```

### 6.2 双方向バインド（Model ↔ View）

```typescript
// テキスト編集 → Model に反映
new go.TextBlock({ editable: true })
  .bind(new go.Binding("text").makeTwoWay())

// 座標の双方向バインド（ドラッグで位置変更 → Model に反映）
new go.Node("Auto", { locationSpot: go.Spot.Center })
  .bind(new go.Binding("location", "loc", go.Point.parse).makeTwoWay(go.Point.stringify))
```

### 6.3 変換関数付き双方向バインド

```typescript
new go.Binding("fill", "temperature",
  // Model → View: 温度に応じた色
  (temp: number) => temp > 30 ? "#F44336" : "#4CAF50"
).makeTwoWay(
  // View → Model（通常は不要だが例として）
  (color: string) => color === "#F44336" ? 35 : 20
)
```

### 6.4 itemArray バインド（リスト表示）

```typescript
new go.Panel("Vertical", {
  itemTemplate: new go.Panel("Horizontal").add(
    new go.TextBlock({ font: "12px sans-serif" }).bind("text", "name"),
    new go.TextBlock({ font: "12px monospace", stroke: "#666" }).bind("text", "type")
  ),
}).bind("itemArray", "columns")

// データ例
// { key: 1, name: "Users", columns: [
//   { name: "id", type: "INT" },
//   { name: "email", type: "VARCHAR" },
// ]}
```

## 8. コンテキストメニュー

```typescript
// ━━━ 右クリックメニュー ━━━
diagram.nodeTemplate = new go.Node("Auto", {
  contextMenu: new go.Adornment("Vertical").add(
    // メニュー項目 1
    makeMenuItem("✏️ 編集", (e, obj) => {
      const node = (obj as go.GraphObject).part as go.Node;
      console.log("編集:", node.data);
    }),
    // メニュー項目 2
    makeMenuItem("📋 コピー", (e, obj) => {
      e.diagram.commandHandler.copySelection();
    }),
    // メニュー項目 3
    makeMenuItem("🗑️ 削除", (e, obj) => {
      e.diagram.commandHandler.deleteSelection();
    }),
  ),
}).add(
  new go.Shape("RoundedRectangle", { fill: "#4CAF50", stroke: null }),
  new go.TextBlock({ margin: 10, stroke: "white" }).bind("text")
);

// メニュー項目のヘルパー関数
function makeMenuItem(
  label: string,
  action: (e: go.InputEvent, obj: go.GraphObject) => void
): go.Panel {
  return new go.Panel("Auto", {
    cursor: "pointer",
    mouseEnter: (e, obj) => {
      const shape = (obj as go.Panel).findObject("BG") as go.Shape;
      if (shape) shape.fill = "#E3F2FD";
    },
    mouseLeave: (e, obj) => {
      const shape = (obj as go.Panel).findObject("BG") as go.Shape;
      if (shape) shape.fill = "white";
    },
    click: action,
  }).add(
    new go.Shape("Rectangle", {
      name: "BG",
      fill: "white",
      stroke: "#ddd",
      strokeWidth: 0.5,
    }),
    new go.TextBlock(label, {
      margin: new go.Margin(6, 12),
      font: "13px sans-serif",
    })
  );
}
```

## 9. 選択時のアドーンメント

```typescript
diagram.nodeTemplate = new go.Node("Auto", {
  // ━━━ 選択時の装飾 ━━━
  selectionAdornmentTemplate: new go.Adornment("Spot").add(
    new go.Placeholder(),
    // グロー効果
    new go.Shape("RoundedRectangle", {
      fill: null,
      stroke: "#2196F3",
      strokeWidth: 3,
      strokeDashArray: [4, 2],
    }),
    // リサイズハンドル
    makeHandle(go.Spot.TopLeft),
    makeHandle(go.Spot.TopRight),
    makeHandle(go.Spot.BottomLeft),
    makeHandle(go.Spot.BottomRight),
  ),
  // リサイズ可能
  resizable: true,
}).add(
  new go.Shape("RoundedRectangle", { fill: "#4CAF50", stroke: null }),
  new go.TextBlock({ margin: 10, stroke: "white" }).bind("text")
);

function makeHandle(spot: go.Spot): go.Shape {
  return new go.Shape("Circle", {
    alignment: spot,
    alignmentFocus: spot,
    cursor: "pointer",
    desiredSize: new go.Size(10, 10),
    fill: "#2196F3",
    stroke: "white",
    strokeWidth: 2,
  });
}
```

## 11. カスタムシェイプ（ジオメトリ）

組み込み図形にない形状は `go.Geometry.parse()` でカスタム定義できます。

```typescript
// ━━━ SVG パス風のカスタムジオメトリ ━━━
go.Shape.defineFigureGenerator("CustomStar", (shape, w, h) => {
  const geo = new go.Geometry();
  const fig = new go.PathFigure(w * 0.5, 0, true); // 開始点
  fig.add(new go.PathSegment(go.SegmentType.Line, w * 0.62, h * 0.38));
  fig.add(new go.PathSegment(go.SegmentType.Line, w, h * 0.38));
  fig.add(new go.PathSegment(go.SegmentType.Line, w * 0.7, h * 0.62));
  fig.add(new go.PathSegment(go.SegmentType.Line, w * 0.8, h));
  fig.add(new go.PathSegment(go.SegmentType.Line, w * 0.5, h * 0.75));
  fig.add(new go.PathSegment(go.SegmentType.Line, w * 0.2, h));
  fig.add(new go.PathSegment(go.SegmentType.Line, w * 0.3, h * 0.62));
  fig.add(new go.PathSegment(go.SegmentType.Line, 0, h * 0.38));
  fig.add(new go.PathSegment(go.SegmentType.Line, w * 0.38, h * 0.38).close());
  geo.add(fig);
  return geo;
});

// 使用
new go.Shape("CustomStar", {
  fill: "#FFD700",
  stroke: "#F57F17",
  desiredSize: new go.Size(50, 50),
})
```

### SVG パスから変換

```typescript
// SVG path 文字列をそのまま使う
const customGeo = go.Geometry.parse("M 0 0 L 50 0 L 50 50 L 0 50 Z", true);
new go.Shape({
  geometry: customGeo,
  fill: "#2196F3",
  stroke: "#1565C0",
})
```

```typescript
// ━━━ テーマ定義 ━━━
const themes = {

    nodeText: "#333333",
    linkColor: "#999999",
    accentColor: "#2196F3",
  },
  dark: {
    background: "#1E1E1E",
    nodeBackground: "#2D2D2D",
    nodeBorder: "#555555",
    nodeText: "#E0E0E0",
    linkColor: "#666666",
    accentColor: "#64B5F6",
  },
};

type Theme = typeof themes.light;

// ━━━ テーマ適用関数 ━━━
function applyTheme(diagram: go.Diagram, theme: Theme): void {
  diagram.startTransaction("theme");

  // 背景色
  diagram.div!.style.backgroundColor = theme.background;

  // 全ノードのスタイルを更新
  diagram.nodes.each((node) => {
    diagram.model.set(node.data, "background", theme.nodeBackground);
    diagram.model.set(node.data, "border", theme.nodeBorder);
    diagram.model.set(node.data, "textColor", theme.nodeText);
  });

  diagram.commitTransaction("theme");
}

// ━━━ 使用例 ━━━
applyTheme(diagram, themes.dark);
```
