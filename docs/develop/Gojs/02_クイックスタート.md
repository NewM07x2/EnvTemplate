## 2. 方法①：CDN で最速スタート（HTML のみ）

**もっとも簡単な方法**です。1つの HTML ファイルだけで GoJS を試せます。

### ステップ 1: HTML ファイルを作成

```html
<!-- index.html -->
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>GoJS クイックスタート</title>

  <!-- GoJS をCDNから読み込み（開発版: go-debug.js） -->
  <script src="https://cdn.jsdelivr.net/npm/gojs/release/go-debug.js"></script>

  <style>
    /* ダイアグラムの表示領域 */
    #myDiagramDiv {
      width: 800px;
      height: 500px;
      border: 1px solid #ccc;
      background-color: #f8f8f8;
    }
  </style>
</head>
<body>
  <h1>🎉 はじめての GoJS ダイアグラム</h1>

  <!-- ダイアグラムをホストする div -->
  <div id="myDiagramDiv"></div>

  <script>
    // Diagram を作成（div の id を指定）
    const myDiagram = new go.Diagram("myDiagramDiv", {
      // Ctrl+Z / Ctrl+Y で Undo/Redo を有効化
      "undoManager.isEnabled": true,
      // TreeLayout で上から下にノードを配置
      layout: new go.TreeLayout({ angle: 90, layerSpacing: 40 }),
    });

    // ノードのテンプレートを定義
    myDiagram.nodeTemplate =
      new go.Node("Auto")
        .add(
          // 背景の角丸矩形
          new go.Shape("RoundedRectangle", {
            fill: "lightblue",
            stroke: "#333",
            strokeWidth: 1.5,
          })
            .bind("fill", "color"),
          // テキスト
          new go.TextBlock({
            margin: 10,
            font: "bold 14px sans-serif",
          })
            .bind("text", "name")
        );

    // リンクのテンプレートを定義
    myDiagram.linkTemplate =
      new go.Link({ routing: go.Routing.Orthogonal, corner: 5 })
        .add(
          new go.Shape({ strokeWidth: 2, stroke: "#555" }),
          new go.Shape({ toArrow: "Standard", stroke: "#555", fill: "#555" })
        );

    // モデルデータを設定（TreeModel: parent で親子関係を定義）
    myDiagram.model = new go.TreeModel([
      { key: 1, name: "CEO",        color: "#FFD700" },
      { key: 2, name: "CTO",        color: "#87CEEB", parent: 1 },
      { key: 3, name: "CFO",        color: "#87CEEB", parent: 1 },
      { key: 4, name: "エンジニア",  color: "#98FB98", parent: 2 },
      { key: 5, name: "デザイナー",  color: "#98FB98", parent: 2 },
      { key: 6, name: "経理",       color: "#DDA0DD", parent: 3 },
    ]);
  </script>
</body>
</html>
```

### ステップ 2: ブラウザで開く

```bash
# ファイルをブラウザで直接開く
# Windows:
start index.html

# Mac:
open index.html
```

### ステップ 3: 操作してみる

| 操作 | 方法 |
|------|------|
| ノードを移動 | ドラッグ＆ドロップ |
| ズーム | マウスホイール |
| パン（全体移動） | 背景をドラッグ |
| 選択 | ノードをクリック |
| 複数選択 | Ctrl + クリック |
| コピー | Ctrl + C → Ctrl + V |
| 元に戻す | Ctrl + Z |
| やり直す | Ctrl + Y |
| 削除 | Delete キー |

## 3. 方法②：npm + TypeScript プロジェクト

### ステップ 1: プロジェクトを作成

```bash
mkdir gojs-demo && cd gojs-demo
npm init -y
```

### ステップ 2: パッケージをインストール

```bash
# GoJS 本体
npm install gojs

# 開発ツール
npm install -D typescript vite
```

### ステップ 3: TypeScript 設定

```json
// tsconfig.json
{
  "compilerOptions": {
    "target": "ES2020",
    "module": "ESNext",
    "moduleResolution": "bundler",
    "strict": true,
    "esModuleInterop": true,
    "outDir": "dist",
    "sourceMap": true
  },
  "include": ["src/**/*"]
}
```

### ステップ 4: ソースコードを作成

```html
<!-- index.html -->
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>GoJS + TypeScript</title>
  <style>
    #myDiagramDiv {
      width: 100%;
      height: 600px;
      border: 1px solid #ccc;
    }
  </style>
</head>
<body>
  <div id="myDiagramDiv"></div>
  <script type="module" src="/src/main.ts"></script>
</body>
</html>
```

```typescript
// src/main.ts
import * as go from "gojs";

function init(): void {
  const diagram = new go.Diagram("myDiagramDiv", {
    "undoManager.isEnabled": true,
    layout: new go.TreeLayout({ angle: 90, layerSpacing: 40 }),
  });

  // ノードテンプレート
  diagram.nodeTemplate =
    new go.Node("Auto")
      .add(
        new go.Shape("RoundedRectangle", {
          fill: "white",
          strokeWidth: 1.5,
          portId: "",
          fromLinkable: true,
          toLinkable: true,
        })
          .bind("fill", "color"),
        new go.TextBlock({
          margin: 8,
          font: "bold 14px sans-serif",
          editable: true, // ダブルクリックでテキスト編集可能
        })
          .bind("text", "name", null, null) // TwoWay バインディングではない
      );

  // リンクテンプレート
  diagram.linkTemplate =
    new go.Link({ routing: go.Routing.Orthogonal, corner: 5 })
      .add(
        new go.Shape({ strokeWidth: 2, stroke: "#555" }),
        new go.Shape({ toArrow: "Standard", fill: "#555", stroke: null })
      );

  // モデル
  diagram.model = new go.TreeModel([
    { key: 1, name: "ルート",     color: "#FFD700" },
    { key: 2, name: "子ノード A", color: "#87CEEB", parent: 1 },
    { key: 3, name: "子ノード B", color: "#87CEEB", parent: 1 },
    { key: 4, name: "孫ノード",   color: "#98FB98", parent: 2 },
  ]);
}

// DOM 読み込み後に初期化
document.addEventListener("DOMContentLoaded", init);
```

### ステップ 5: 起動

```json
// package.json に追加
{
  "scripts": {
    "dev": "vite",
    "build": "vite build"
  }
}
```

```bash
npm run dev
# → http://localhost:5173 で開く
```

### ステップ 3: ダイアグラムコンポーネントを作成

```tsx
// src/components/OrgChart.tsx
import { useCallback, useState } from "react";
import * as go from "gojs";
import { ReactDiagram } from "gojs-react";
import "./OrgChart.css";

// Diagram の初期化関数（一度だけ呼ばれる）
function initDiagram(): go.Diagram {
  const diagram = new go.Diagram({
    "undoManager.isEnabled": true,
    layout: new go.TreeLayout({
      angle: 90,
      layerSpacing: 40,
      nodeSpacing: 20,
    }),
    model: new go.TreeModel({
      nodeKeyProperty: "key",
      nodeParentKeyProperty: "parent",
    }),
  });

  // ノードテンプレート
  diagram.nodeTemplate = new go.Node("Auto", {
    locationSpot: go.Spot.Center,
  })
    .add(
      new go.Shape("RoundedRectangle", {
        fill: "white",
        stroke: "#333",
        strokeWidth: 1.5,
      }).bind("fill", "color"),
      new go.TextBlock({
        margin: 10,
        font: "bold 14px sans-serif",
        editable: true,
      }).bind("text", "name")
    );

  // リンクテンプレート
  diagram.linkTemplate = new go.Link({
    routing: go.Routing.Orthogonal,
    corner: 5,
  }).add(
    new go.Shape({ strokeWidth: 2, stroke: "#555" }),
    new go.Shape({ toArrow: "Standard", fill: "#555", stroke: null })
  );

  return diagram;
}

export default function OrgChart() {
  const [nodeDataArray] = useState([
    { key: 1, name: "CEO", color: "#FFD700" },
    { key: 2, name: "CTO", color: "#87CEEB", parent: 1 },
    { key: 3, name: "CFO", color: "#87CEEB", parent: 1 },
    { key: 4, name: "エンジニア", color: "#98FB98", parent: 2 },
    { key: 5, name: "デザイナー", color: "#98FB98", parent: 2 },
    { key: 6, name: "経理", color: "#DDA0DD", parent: 3 },
  ]);

  const handleModelChange = useCallback((e: go.IncrementalData) => {
    // ユーザーの操作による変更をここで受け取る
    console.log("Model changed:", e);
  }, []);

  return (
    <ReactDiagram
      initDiagram={initDiagram}
      divClassName="diagram-component"
      nodeDataArray={nodeDataArray}
      onModelChange={handleModelChange}
    />
  );
}
```

```css
/* src/components/OrgChart.css */

  width: 100%;
  height: 600px;
  border: 1px solid #ccc;
  background-color: #f8f8f8;
}
```

```tsx
import { useCallback, useState } from "react";

import { ReactDiagram } from "gojs-react";

function initDiagram(): go.Diagram {

  diagram.nodeTemplate = new go.Node("Auto")
    .add(
      new go.Shape("RoundedRectangle", {

        stroke: "#333",

  diagram.linkTemplate = new go.Link({

export default function DiagramWrapper() {
  const [nodeData] = useState([
    { key: 1, name: "CEO",   color: "#FFD700" },
    { key: 2, name: "CTO",   color: "#87CEEB", parent: 1 },

```

```tsx
// src/app/page.tsx
import dynamic from "next/dynamic";

// SSR を無効化してクライアントサイドでのみレンダリング
const DiagramWrapper = dynamic(
  () => import("@/components/DiagramWrapper"),
  { ssr: false }
);

export default function Home() {
  return (
    <main>
      <h1>GoJS + Next.js デモ</h1>
      <DiagramWrapper />
    </main>
  );
}
```

> **⚠️ 重要:** GoJS は DOM（Canvas）を使用するため、Next.js では必ず `ssr: false` で dynamic import してください。

```vue
      }).bind("text", "name")
    );

  // リンクテンプレート
  diagram.linkTemplate = new go.Link({
    routing: go.Routing.Orthogonal,
    corner: 5,
  }).add(
    new go.Shape({ strokeWidth: 2, stroke: "#555" }),
    new go.Shape({ toArrow: "Standard", fill: "#555", stroke: null })

    { key: 1, name: "CEO",       color: "#FFD700" },
```

## 7. 最初のダイアグラムを理解する

### コードの構造

```typescript
// 1️⃣ Diagram の作成
const diagram = new go.Diagram("myDiagramDiv", {
  "undoManager.isEnabled": true,      // Undo/Redo を有効化
  layout: new go.TreeLayout({ ... }), // 自動レイアウト
});

// 2️⃣ ノードテンプレートの定義
diagram.nodeTemplate =
  new go.Node("Auto")                // Panel タイプ: Auto
    .add(
      new go.Shape("RoundedRectangle", { ... }),  // 背景図形
      new go.TextBlock({ ... })                    // テキスト
        .bind("text", "name")                      // データバインド
    );

// 3️⃣ リンクテンプレートの定義
diagram.linkTemplate =
  new go.Link({ ... })
    .add(
      new go.Shape({ ... }),           // 線
      new go.Shape({ toArrow: "Standard" })  // 矢印
    );

// 4️⃣ モデルデータの設定
diagram.model = new go.TreeModel([
  { key: 1, name: "親ノード" },
  { key: 2, name: "子ノード", parent: 1 },
]);
```

### データバインディングの仕組み

```
Model データ:                    ノードの表示:
{ name: "Alice", color: "red" }
     │                           ┌──────────────┐
     │  .bind("text", "name")    │ ┌──────────┐ │
     ├──────────────────────────→│ │  Alice   │ │ ← TextBlock.text = "Alice"
     │                           │ └──────────┘ │
     │  .bind("fill", "color")   │  Background  │
     └──────────────────────────→│  ■■ red ■■  │ ← Shape.fill = "red"
                                 └──────────────┘
```

## 8. データの保存と復元（JSON）

GoJS のモデルは JSON として保存・復元できます。

### 保存

```typescript
// Model を JSON 文字列として取得
const jsonString = diagram.model.toJson();
console.log(jsonString);
// → {"class":"go.TreeModel","nodeDataArray":[{"key":1,"name":"CEO",...},...]}

// ローカルストレージに保存
localStorage.setItem("diagramData", jsonString);

// サーバーに送信
await fetch("/api/diagram", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: jsonString,
});
```

### 復元

```typescript
// JSON 文字列からモデルを復元
const jsonString = localStorage.getItem("diagramData");
if (jsonString) {
  diagram.model = go.Model.fromJson(jsonString);
}
```

## 9. デバッグのコツ

### go-debug.js を使う

開発中は `go-debug.js`（デバッグ版）を使用してください。エラーメッセージがより詳細になります。

```html
<!-- 開発時 -->
<script src="https://cdn.jsdelivr.net/npm/gojs/release/go-debug.js"></script>

<!-- 本番時 -->
<script src="https://cdn.jsdelivr.net/npm/gojs/release/go.js"></script>
```

npm の場合は自動的に適切なバージョンが選択されます。

### モデルデータの確認

```typescript
// 現在のノードデータを確認
console.log(diagram.model.nodeDataArray);

// 特定のノードデータを取得
const nodeData = diagram.model.findNodeDataForKey(1);
console.log(nodeData);
```

### Diagram にアクセスするグローバル変数

```typescript
// デバッグ用にグローバルに公開
(window as any).myDiagram = diagram;

// ブラウザのコンソールから:
// myDiagram.nodes.count → ノード数
// myDiagram.selection.first() → 選択中のノード
```
